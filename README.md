# Amabam

![Status](https://img.shields.io/badge/Status-Active-brightgreen?style=flat-square)
![HTML5](https://img.shields.io/badge/HTML5-Redirect%20Page-E34F26?style=flat-square&logo=html5)

Amabam is a platform accessible at [amabam.replit.app](https://amabam.replit.app/signup). This repository hosts the GitHub Pages redirect page that routes visitors from the GitHub URL to the live application.

---

## Features

- Animated dark loading screen while redirect processes
- Automatic redirect to the live Amabam app
- Zero-dependency static HTML

## How It Works

When a user visits this GitHub Pages URL, an animated splash page displays briefly before redirecting to the live app hosted on Replit. The redirect uses an HTML meta refresh tag for instant compatibility.

## How to Run Locally

```bash
git clone https://github.com/danielmiranda24b/Amabam.git
cd Amabam
# Open index.html in your browser
```
